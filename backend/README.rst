Installation
------------

To install fedmix-backend, do:

.. code-block:: console

  git clone https://github.com/FEDMix/fedmix-backend.git
  cd fedmix-backend
  pip install .

To run fedmix-backend, do:

.. code-block:: console

  fedmix-backend


Run tests (including coverage) with:

.. code-block:: console

  python setup.py test
